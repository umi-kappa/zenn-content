## はじめに

はじめまして、 [Kurashita](https://x.com/umi_kappa) です。

以前は、 [Qiita](https://qiita.com/umi_kappa) で記事を書いていたのですが、子供が生まれ、仕事もプライベートも忙しくなり、なかなかアウトプットに使う時間（気力）が作れない日々でした。

そんな中、子育ても少しずつ慣れてきたこともあり、自分の時間も少しずつですが取れるようになってきて、そろそろエンジニアとしてアウトプットしないとな〜とぼんやり考えては見たものの、重い腰は上がらず😅

その状態の僕の背中を押してくれたのはAIでした。
以前は記事を書くにも、構成を考えるところから文章を書き、書き上がった文章がおかしくないかを一人でチェックする必要があったりと、思い立ってから公開まで長い道のりでした。
そこをほぼほぼAIがやってくれるという...ありがとうAI...子育てでクタクタな自分でも記事が書けるよありがとう😭

ということで、これからAIに手伝ってもらいながら、日々の開発で学んだことや、気になった技術について発信していきたいと思います！（ちゃんと続けていかねば）

最初の記事として、まずは自己紹介と普段使っている技術について書きます。

### Claude Code / AI 活用

最近は Claude Code を開発ワークフローに取り入れています。

- GitHub Copilot はエディタの補完で活用していました
  - 最近はあまり使わず
- 主にコードを書くときは Cursor を使っています
  - これも最近頻度が減った
- Claude Code は 2026 年に入ってから使い始めました
  - 業務ではほぼ毎日使っています。もう使わない日々を思い出せないくらい
  - といいつつ、まだ AI を完全に信じていないので、自分の目でのレビューはしっかり目にやっています
  - 最近は他メンバーのレビューの数も増えてきたので、 CodeRabbit だったり、 Claude Code にレビューを手伝ってもらっています

ちなみにこの記事自体も、Claude Code に初稿を書いてもらって、不足しているなと思うところを追加して書いています。 plan ファイルも [GitHub](https://github.com/umi-kappa/zenn-content) に commit してみたので、興味ある方はぜひ。
